# Template ของหนังสือบันทึกข้อความราชการ

## โครงสร้างเอกสาร (A4)

### 1. ส่วนหัว (Header)
```
[ตราครุฑไทย]  <- ตรงกลาง, ขนาด 60x60px
ตำแหน่ง: กลางหน้า ห่างจากขอบบน 2cm

[ชื่อหน่วยงาน]  <- THSarabunNew Bold, 16pt, ตรงกลาง
เช่น: "สำนักงานพัฒนาธุรกรรมทางอิเล็กทรอนิกส์"

[ชั้นความเร็ว]  <- ถ้ามี (ด่วนที่สุด/ด่วนมาก/ด่วน)
THSarabunNew Regular, 14pt, ตรงกลาง
```

### 2. ส่วนรายละเอียดหนังสือ
```
ที่: [เลขที่หนังสือ]         วันที่: [วันที่ภาษาไทย]
เช่น: "ที่ กค 0103/2568"    "29 ธันวาคม พ.ศ. 2568"

เรื่อง: [หัวเรื่องหนังสือ]  <- THSarabunNew Bold
เช่น: "ขอเชิญประชุม"

เรียน: [ผู้รับหนังสือ/ตำแหน่ง]
เช่น: "ผู้อำนวยการกองนโยบายและแผน"
```

### 3. เนื้อหาเอกสาร
```
     [เนื้อหา paragraph 1]
indent ประมาณ 50px จากขอบซ้าย
justify text
THSarabunNew Regular, 14pt
บรรทัดห่าง 1.5

รองรับ:
- หัวข้อย่อย (bookContentTitle)
- เนื้อหาแต่ละส่วน (bookContent)
- การขึ้นบรรทัดใหม่ (\n)
- รายการแบบ bullet points
```

### 4. ลายเซ็น (ขวาล่าง)
```
                              (ลายเซ็นจริง - ว่างไว้)
                              
                              (นายสมชาย  ใจดี)
                              ผู้อำนวยการสำนักงาน
                              
ระยะห่างจากขอบขวา: ~200px
THSarabunNew Regular, 14pt
```

## กรณีพิเศษ

### กรณีมีหลายประเภทลายเซ็น (case5_with_all_relations)
```
ผู้ลงนาม (bookSigned):     - แสดงข้างล่างสุด ทางขวา
                           - prefix + firstname + lastname
                           - ตำแหน่ง (positionName)

ผู้เสนอผ่าน (bookSubmited): - แสดงเหนือผู้ลงนาม
ผู้ทราบ (bookReview):       - ถ้าต้องการแสดง
ผู้รับ (bookLearner):       - แสดงในส่วน "เรียน"
```

### รูปแบบกระดาษ
```
A4 (210mm x 297mm)
Margin:
  - Top: 70pt (~2.5cm)
  - Bottom: 50pt
  - Left: 50pt (~1.8cm)
  - Right: 50pt
```

## ภาพตัวอย่าง

รูปโลโก้ที่ใช้:
- `logoThaiGov.png` - ตราครุฑไทย (ทั่วไป)
- `logoETDA.png` - โลโก้ ETDA
- `logoMDES.png` - โลโก้กระทรวงดิจิทัลฯ
- `etda_stamp_red.png` - ตราประทับ ETDA สีแดง
- `mdes_stamp_red.png` - ตราประทับ MDES สีแดง

## ฟอนต์ที่ใช้

THSarabun New (12 ไฟล์):
- THSarabunNew.ttf (Regular)
- THSarabunNew Bold.ttf
- THSarabunNew BoldItalic.ttf
- THSarabunNew Italic.ttf
... และอื่นๆ

ขนาดฟอนต์:
- หัวข้อหลัก: 16pt Bold
- หัวข้อรอง: 14pt Bold  
- เนื้อหา: 14pt Regular
- ลายเซ็น: 14pt Regular

## จุดสำคัญ

1. **ตราครุฑ**: ต้องอยู่กลางหน้า ห่างจากขอบบน 2cm
2. **ชื่อหน่วยงาน**: ตรงกลาง ตัวหนา
3. **วันที่**: รูปแบบไทย "29 ธันวาคม พ.ศ. 2568"
4. **เนื้อหา**: indent แรก justify text
5. **ลายเซ็น**: ขวาล่าง ห่างขอบขวา 200px
6. **รองรับภาษาไทย**: ต้องใช้ฟอนต์ THSarabunNew

## Test Cases ที่ควรทดสอบ

1. case1_basic_memo - บันทึกข้อความพื้นฐาน
2. case2_with_learner - มีผู้รับ (เรียน)
3. case3_with_signature - มีลายเซ็น 1 คน
4. case4_multi_pages - เนื้อหายาว หลายหน้า
5. case5_with_all_relations - มีครบทุกประเภท (ลงนาม, เสนอผ่าน, ทราบ, ผู้รับ)
6. case6_minimal - ข้อมูลน้อยสุด
7. case7_thai_special_chars - อักษรไทยพิเศษ และสัญลักษณ์
